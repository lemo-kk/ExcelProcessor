一、整体设计风格

主题风格

暗色系为主（科技感，避免长时间盯屏疲劳），搭配高亮色点缀。

背景：深蓝 / 深灰（#1E1E2F / #121212）

点缀色：赛博蓝 (#00E5FF) / 荧光绿 (#3DFF92) / 高亮紫 (#9D4EDD)

提示和错误：橙色 (#FFB347)、红色 (#FF4D4F)

布局规范

一级菜单：上方横向导航栏，悬浮高亮。

二级功能区：左侧树形/标签式菜单（可选）。

主要工作区：中间卡片式布局（表格、配置区、可视化执行日志）。

右侧/底部：日志 & 执行进度区。

字体 & 图标

字体：Segoe UI / 微软雅黑（12-14px，标题加粗 16-18px）。

图标：线性科技风（推荐 Lucide / Remixicon）。

按钮：圆角 8px / hover 发光效果。

📑 二、一级菜单（顶部导航栏）
[ 首页 ] [ Excel导入配置 ] [ SQL管理 ] [ 作业配置 ] [ 数据源管理 ] [ 导入导出配置 ] [ 系统设置 ]

统一悬浮样式：文字变亮 + 底部发光下划线。

激活态：高亮蓝色背景，字体加粗。

已实现功能：
- 主窗口导航：MainWindow.xaml
- 页面切换：Frame控件实现
- 导航按钮：Button控件，支持Click事件
- 当前页面高亮：通过IsSelected属性控制

🖼️ 三、页面设计规范
1. 首页

风格：仪表盘式，高科技面板。

模块：

快捷入口卡片（EXCEL导入、作业配置）

最近执行任务列表（卡片 + 状态标签：成功/失败/运行中）

系统资源（SQLite大小、任务数量）

2. Excel导入配置

表格样式：

统一表格主题：深灰底，表头高亮蓝色，悬浮行变色。

表格行操作按钮：📝 编辑 | ❌ 删除（悬浮显示）

表单样式：

左对齐，标签右对齐（如“文件路径：”）

输入框：深色圆角矩形，获取焦点时边框发蓝光

3. SQL管理

三列布局：
- 左侧：SQL列表管理
- 中间：SQL详情编辑  
- 右侧：SQL处理配置（执行参数、参数配置、操作按钮）

代码编辑器风格：

高科技风，深色代码区，语法高亮（蓝色关键词，绿色字符串）。

支持折叠、自动提示。

执行按钮：右下角固定，主按钮"立即执行"（蓝色发光大按钮）。

4. 作业配置

表格列表（作业ID、作业名称、调度模式、一键执行按钮）。

支持：

新建作业 → 弹窗配置（选择 Excel源、SQL处理链路、输出配置）。

定时任务配置 → Cron表达式输入 + 图形化时间选择器。

一键执行 → 行内按钮，点击后弹出执行进度条。

5. 数据源管理

数据库列表（默认数据库 + 自定义数据源）。

操作：添加/修改/测试连接。

样式：卡片式展示，每个数据库卡片悬浮时发光。

6. 导入导出配置

列表表格：保存过的配置方案（方便客户复用）。

操作：导入方案（上传 JSON/XML 配置）、导出方案。

7. 系统设置

卡片分区：

用户管理

日志级别（Info/Debug/Error）

路径配置（相对路径/绝对路径切换）

UI元素统一：开关（Switch风格）、下拉框统一圆角。

🔘 四、控件规范

按钮

主按钮（执行类）：蓝色渐变 + 光晕

次按钮（配置类）：灰色 + hover变亮

危险按钮（删除类）：红色

输入框

圆角 6px，背景深灰，输入时蓝色发光边框

表格

表头深蓝底，白色文字

行悬浮变亮

行内操作按钮浮动在最右侧

日志区

黑底 + 彩色文本（成功：绿色，失败：红色，警告：橙色）

🧩 五、交互亮点

所有按钮 hover 时带 微弱的光晕扩散动画。

切换菜单时有 滑动过渡效果。

作业执行进度用 环形进度条 + 动态数字，科技感更强。
